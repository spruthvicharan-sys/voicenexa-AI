Voicenexa AI

Real-time voice AI assistant that listens, thinks, and speaks back.

Live Demo: https://voicenexaai.site
GitHub: https://github.com/spruthvicharan-sys/voicenexa-AI

Features
Real-time voice conversations
Voice Activity Detection
Whisper speech to text
Groq AI responses powered by Llama 3.3 70B
ElevenLabs voice synthesis
Multi-turn conversation memory
Wake word detection Hey Nexa
3D animated voice orb
Fully responsive design

Tech Stack
Frontend: React, Vite, Tailwind CSS, Three.js, Framer Motion
Backend: Python, FastAPI, WebSockets
Speech to Text: OpenAI Whisper
AI Intelligence: Groq API Llama 3.3 70B
Voice Synthesis: ElevenLabs
Deployment: Docker, AWS EC2, Nginx

Setup
1. Clone the repository
2. Copy .env.example to .env and add your API keys
3. Run docker compose up then build
4. Open http://localhost

Required API Keys
GROQ_API_KEY from console.groq.com
ELEVENLABS_API_KEY from elevenlabs.io
ELEVENLABS_VOICE_ID from your ElevenLabs dashboard
